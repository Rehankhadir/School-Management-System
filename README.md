"# School Management System Frontend

A modern React + Vite + Tailwind CSS frontend template for a school management system. This project includes common dashboard pages for students, teachers, attendance, fees, reports, and more.

## Features

- React 19 with Vite for fast development and production builds
- Tailwind CSS for utility-first styling
- React Router v7 for page-based navigation
- Dashboard layout with protected routes and role-based pages
- Demo pages for students, teachers, attendance, marks, notifications, reports, and more
- Mock data support for quick prototyping
- Accessible UI components and responsive design

## Project structure

- `src/`
  - `components/` – shared UI components
  - `context/` – authentication and app state context
  - `data/` – mock data files
  - `hooks/` – reusable hooks
  - `layouts/` – common page layouts
  - `pages/` – page views for the dashboard
  - `routes/` – route guards and protected route components

## Getting started

### Prerequisites

- Node.js 20+ or compatible version
- npm 10+ (or yarn/pnpm if configured)

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

Open the local Vite URL shown in the terminal to view the app.

### Build for production

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```

## Package scripts

- `dev` – start the Vite development server
- `build` – build the app for production
- `preview` – preview the production build locally

## Notes

This frontend is designed as a template for an educational dashboard. It can be extended with real backend authentication, API integration, and data persistence as needed.
"
