# School Management Mobile App

This is a separate mobile-first version of the existing school management frontend. The original web application remains unchanged.

## What it includes

- Role-based quick login for admin, teacher, student, parent, and accountant
- Mobile dashboard with role-aware shortcuts
- Students, student profile details, teachers, attendance, marks, exams, timetable, fees, leaves, announcements, notifications, reports, and profile screens
- Touch-first UI patterns: bottom navigation, module drawer, compact cards, filters, and bottom sheets
- Shared mock data imported from the existing web app source

## Run locally

```bash
npm run dev
```

The dev server defaults to port `5174`.

## Build

```bash
npm run build
```
